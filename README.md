## What AutoDocify Does

AutoDocify is a project designed to automate the process of generating documentation.  It streamlines the creation of documentation, reducing manual effort and ensuring consistency.  This is a version **1.0** of **AutoDocify** and this **ReadMe** is aslo generated by the **AutoDocify**.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/dev-sharma16/AutoDocify.git
   ```
2. Navigate to the project directory:
   ```bash
   cd AutoDocify
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
   or
   ```bash
   yarn install 
   ```

## Running the Project

### Development

To run the project in development mode:

```bash
npm run dev
```

This will start a development server and open the application in your browser.


### Production

To build the project for production:

```bash
npm run build
```

This will create an optimized build in the `dist` directory.  You can then deploy this build to your server.
